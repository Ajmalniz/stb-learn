Chapter 7 · Topic 1

Oxidation & Reduction

Oxidation and reduction always happen together — you can never have one without the other. Together they are called redox reactions, and they underpin electrochemistry, corrosion, batteries, respiration, and almost every important chemical process in life and industry.

1.1 What is Oxidation?

Historically, oxidation meant combining with oxygen. The modern, broader definition is based on electron transfer.

Oxidation (modern definition)
Loss of electrons by an atom or ion. The oxidation state increases.
Oxidation (classical definition)
Gain of oxygen OR loss of hydrogen by a substance.
Oxidising Agent
The substance that accepts electrons from another. It causes the other substance to be oxidised, but is itself reduced in the process. Examples: O₂, Cl₂, KMnO₄, concentrated H₂SO₄.

1.2 What is Reduction?

Reduction (modern definition)
Gain of electrons by an atom or ion. The oxidation state decreases.
Reduction (classical definition)
Loss of oxygen OR gain of hydrogen by a substance.
Reducing Agent
The substance that donates electrons to another. It causes the other substance to be reduced, but is itself oxidised in the process. Examples: Zn, C (coke), CO, H₂, Na.

1.3 The OIL RIG Memory Trick

Memory Trick — OIL RIG

OIL RIG

Oxidation Is Loss  ·  Reduction Is Gain

This refers to the loss or gain of electrons. Once you remember OIL RIG, you will never confuse oxidation and reduction again.

Property Oxidation Reduction
ElectronsLost (OIL)Gained (RIG)
Oxidation stateIncreasesDecreases
OxygenGainedLost
HydrogenLostGained

1.4 Oxidation States (Oxidation Numbers)

The oxidation state is a number we assign to an atom to keep track of electron changes. It is not a real charge — it is a bookkeeping tool.

Rules for Assigning Oxidation States
  1. The oxidation state of a free element (uncombined) = 0. (e.g., Fe, O₂, Cl₂ all = 0)
  2. The oxidation state of a monoatomic ion equals its ionic charge. (e.g., Na⁺ = +1; Cl⁻ = −1; Fe³⁺ = +3)
  3. Oxygen is usually −2 (except in peroxides like H₂O₂ where it is −1, and OF₂ where it is +2).
  4. Hydrogen is usually +1 (except in metal hydrides like NaH where it is −1).
  5. The sum of oxidation states in a neutral compound = 0.
  6. The sum of oxidation states in a polyatomic ion = the ionic charge of that ion.

Example — finding the oxidation state of S in H₂SO₄:
H₂SO₄ is neutral (sum = 0). H = +1, O = −2.
2(+1) + x + 4(−2) = 0 → 2 + x − 8 = 0 → x = +6

Example — finding the oxidation state of Mn in MnO₄⁻:
MnO₄⁻ has charge −1. O = −2.
x + 4(−2) = −1 → x − 8 = −1 → x = +7

1.5 Worked Examples of Redox Reactions

Example 1 — Zinc and Copper Sulphate:

Zn(s) + CuSO₄(aq) → ZnSO₄(aq) + Cu(s)
  • Zn: 0 → +2  (oxidation state increases → oxidised). Zn is the reducing agent.
  • Cu²⁺: +2 → 0  (oxidation state decreases → reduced). Cu²⁺ is the oxidising agent.
  • Half-reaction (oxidation): Zn → Zn²⁺ + 2e⁻
  • Half-reaction (reduction): Cu²⁺ + 2e⁻ → Cu

Example 2 — Burning Magnesium:

2Mg(s) + O₂(g) → 2MgO(s)
  • Mg: 0 → +2 → oxidised (Mg is the reducing agent)
  • O: 0 → −2 → reduced (O₂ is the oxidising agent)

Example 3 — Blast Furnace (Reduction of Iron Ore):

Fe₂O₃(s) + 3CO(g) → 2Fe(l) + 3CO₂(g)
  • Fe: +3 → 0 → reduced (Fe₂O₃ is the oxidising agent)
  • C in CO: +2 → +4 → oxidised (CO is the reducing agent)
Did you know?

In the blast furnace, coke (carbon) burns to form CO₂, which then reacts with more coke to form CO. It is the CO that actually reduces the iron oxide — not the carbon directly! CO is the true reducing agent in iron extraction.
