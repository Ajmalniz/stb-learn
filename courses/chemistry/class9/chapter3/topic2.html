Chapter 3 · Topic 2

Periods, Groups & Blocks

The modern periodic table has a very logical structure. Once you understand periods (rows), groups (columns), and blocks (sections), you can read the table like a map and instantly know a lot about any element.

2.1 Periods (Horizontal Rows)

Period
A horizontal row of elements in the periodic table. The period number equals the number of electron shells in the atoms of that period.
Period No. of Shells No. of Elements Elements
112H, He
228Li to Ne
338Na to Ar
4418K to Kr (includes transition metals)
5518Rb to Xe
6632Cs to Rn (includes lanthanides)
7732Fr to Og (includes actinides)

As you move across a period, the atomic number increases by 1 each step. All elements in a period have the same number of shells, but different numbers of valence electrons.

2.2 Groups (Vertical Columns)

Group
A vertical column of elements. Elements in the same group have the same number of valence electrons → similar chemical properties.

2.3 Blocks of the Periodic Table

The periodic table is divided into four blocks based on which sub-shell the last electron fills into:

s-block
Groups 1 & 2. Last electron fills an s orbital. Includes alkali metals and alkaline earth metals. Very reactive metals (except H and He).
p-block
Groups 13–18. Last electron fills a p orbital. Contains metals, non-metals, and metalloids. Includes the halogens and noble gases.
d-block
Groups 3–12. Last electron fills a d orbital. These are the transition metals (Fe, Cu, Zn, etc.). Many form coloured compounds and have variable oxidation states.
f-block
Placed below the main table. Last electron fills an f orbital. Includes Lanthanides (Period 6, Ce–Lu) and Actinides (Period 7, Th–Lr). Actinides include radioactive elements like uranium and plutonium.
